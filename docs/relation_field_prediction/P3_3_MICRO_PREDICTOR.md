# P3-3 ミクロ力学予測本体

## 目的

P3-2で翻訳した共通ミクロ流れ情報から、1ステップ先の辺流れと分布`G_{t+1}`を実際に予測する。

P3-3は試作機であり、完成版ではない。初回は次の3候補を同時に出し、P3-5で精度と条件別バランスを比較する。

```text
A 継続予測
B 加速度補正予測
C 局所結合補正予測
```

## 入力境界

使用するもの：

- 正本`G_t`
- `cutoff_t`までの因果的`K_t`
- RF-2固定格子
- P3-2共通ミクロ流れ情報

使用しないもの：

- 疑似現実系の内部状態
- 未来状態・未来外力・作用予定
- scenario、seed、split
- RF-10
- truth・outcome
- P1・P2

## 予測A：現在流れ継続

各固定辺について、現在代表流量が次の1ステップでも続くとする。

```text
F_A(t+1) = F_t
```

これは最小の流れ予測基準である。

## 予測B：加速度補正

P3-2の現在流量と直前流量から、流量変化を使う。

```text
ΔF_t = F_t - F_(t-1)
F_B(t+1) = F_t + α × clip(ΔF_t)
```

初期値：

```text
α = 0.5
```

変化量の上限は、現在流量・候補幅・最小数値幅から作る局所尺度の2倍とする。

これは完成値ではない。P3-5の精度・バランス検証後に修正する。

## 予測C：局所結合補正

予測Bへ、固定格子上で直接つながる活性流れの局所文脈を加える。

支持流れ：

- 対象流れの出発点へ入る流れ
- 対象流れの到着点から先へ続く流れ

競合流れ：

- 同じ出発点から別方向へ出る流れ

```text
局所結合量 = 支持流量平均 - 競合流量平均
F_C(t+1) = F_B(t+1) + β × 局所結合量
```

初期値：

```text
β = 0.25
```

局所結合補正は、現在流量・加速度補正後流量・候補幅から作る尺度の50%以内へ制限する。

RF-8の結合行列は因果係数ではないため、初回試作では数値係数として直接掛けない。P3-2成果物には文脈として保持し、必要性はP3-5後に判断する。

## 新しい流れの扱い

初回試作では、P3-2に存在しない新しい辺流れを直接発生させない。

予測対象は、P3-2で活性または候補として現れた固定辺に限定する。

未観測の新規駆動は、既存流れへ勝手に配分せず、P3-2の新規駆動候補・残差として保持する。

## 分布生成

各予測流れを現在分布へ適用する。

```text
予測G_(t+1) = G_t + 流入 - 流出
```

正の辺流量はRF-2の正方向、負の辺流量は逆方向として扱う。

## 制約

### 過剰流出

同じセルから出る予測流量の合計が現在保有量を超えた場合、同じ出発セルの流れを比例縮小する。

```text
縮小率 = 現在保有量 / 予測総流出量
```

これにより流れ間の比率を維持しながら、負の分布を防ぐ。

### 必須条件

- 負の分布を作らない
- 出発セルの保有量を超えない
- 総質量を維持する
- 制約前後の流量を保存する
- 制約による削減量を保存する
- 正本G/Kへ書き戻さない

## 出力

```text
<output>/
  contract.json
  identity.json
  predictions.npz
  flow_predictions.jsonl
  summary.json
  validation.json
  provenance.json
```

### predictions.npz

- A/B/Cの制約前辺流量
- A/B/Cの制約後辺流量
- A/B/Cの予測分布変化
- A/B/Cの予測`G_(t+1)`
- 加速度寄与
- 局所支持・競合
- 局所結合寄与
- 辺ごとの制約縮小率

### flow_predictions.jsonl

P3-2の各活性固定辺について、A/B/Cの流量予測と各補正の寄与を保存する。

## 評価との分離

P3-3では予測を生成するが、実現未来との比較は行わない。

```text
accuracy_evaluated = false
```

実現未来生成、誤差計算、精度・バランス評価はP3-4・P3-5で行う。

## 初期パラメーター

```text
加速度重み               0.5
加速度上限比             2.0
局所結合重み             0.25
局所結合上限比           0.5
予測時間幅               1ステップ
```

これらは試作機の初期値であり、理論上の最適値ではない。

## P3-3完了境界

完了するもの：

- A/B/Cの1ステップ先辺流れ予測
- 加速度補正の上限制御
- 局所支持・競合による結合補正
- 過剰流出の比例縮小
- 非負・質量保存の未来G生成
- 各補正・制約の追跡

行わないもの：

- 疑似現実系の実現未来生成
- 予測と実現の比較
- 精度・バランス評価
- 長期予測
- 新規辺の発生予測
- 学習器の導入
- 最適輸送・DMD・Koopmanの導入
