# Task 3.2-4 範囲
## マクロ力学抽出の最小試作

## 1. 目的

生状態`X_t`と履歴`L_t`から低次元マクロ状態と時間発展を抽出し、Task 3.2-3の単純基準を未知の外乱予定で超えられるか確認する。

Task 4は最終的な動的関係場や力学分解を固定する段階ではない。

## 2. 難化コーパス

- 6条件群
- seed 10〜14
- 1軌道64遷移
- 合計30軌道、1,920遷移、1,950状態

seedごとに次を変える。

- 外乱開始時刻
- 外乱持続時間
- 外乱強度
- 外力の組合せ
- 残留負荷
- 再悪化の有無と時刻
- 急激な崩壊か蓄積型崩壊か

同じ条件群でも予定表を共通化しない。

分割:

```text
fit: seed 10, 11, 12
validation: seed 13
holdout: seed 14
```

## 3. Task 3比較基準

### 固定元モデル

Task 3の探索コーパスを再生成し、Task 3で固定した構造をseed 0〜3で学習して難化データへ適用する。

### 公平再学習基準

難化コーパスのfitデータで、Task 3の固定構造だけを再学習する。

```text
history_logistic__W08__H04
```

- 履歴幅8
- 予測時間幅4
- 警報閾値0.2
- 警報継続1

Task 4の主比較対象は公平再学習基準である。

## 4. 入力範囲

### 分布のみ

`distribution`を平方根変換し、fitだけで標準化・主成分分析を行う。

### 全状態場

Task 1で固定した23状態配列をすべて使用する。

各配列についてfitデータだけで平均・標準偏差を求め、固定ランダム射影で12成分へ落としてから連結し、fitだけで主成分分析を行う。

巨大な全配列をそのまま分類器へ渡さない。

## 5. 低次元数

```text
4
8
12
```

低次元基底はfitだけで学習し、validationとholdoutは変換だけを行う。

## 6. 比較する力学

### 静的低次元状態

時間発展を持たない低次元比較基準。

### DMD

```text
M_{t+1} ≈ A M_t
```

### DMDc

```text
M_{t+1} ≈ A M_t + B u_t
```

未来外力の実値は使わず、現在観測値が継続すると仮定する。

### 遅延残差力学

履歴幅8のHankel遅延座標を主成分へ落とし、次の低次元状態を予測する。

HAVOKの考え方を使う最小試作であり、残差成分を外力・ノイズ・未知力学のいずれかへ断定しない。

## 7. 抽出する特徴

- 低次元座標
- 低次元状態ノルム
- 低次元変化量
- 未来低次元予測
- 予測変化量
- 予測軌道長
- 一歩予測残差
- 残差平均
- 残差傾き
- DMDc内部応答
- DMDc外力応答
- 内部応答と外力応答の比
- 遅延座標

モードには`mode_01`等の中立名を使い、物理的意味を固定しない。

## 8. 予測比較

各候補について次を比較する。

1. マクロ特徴のみ
2. Task 3特徴＋マクロ特徴

予測時間幅:

```text
H = 4, 8, 16
```

## 9. 近似現在状態・異未来監査

Task 3の現在状態特徴が近い異軌道の時点対を作る。

未来リスク深度差が一定以上ある対について、予測深度の大小関係を正しく区別できた割合を測る。

対が不足する場合は不足を記録し、無理に生成しない。

## 10. 残差台帳

各時点で次を保存する。

```text
観測低次元状態
−
一歩予測低次元状態
=
未説明残差
```

保存項目:

- 残差ノルム
- 危険開始までの残り時間
- 条件群
- seed
- split
- 力学候補
- 低次元数

残差を既知力学へ強制吸収しない。

## 11. 候補選択

validationでは次の辞書順で選択する。

1. 高リスク軌道検出率
2. 警報先行時間
3. 誤警報率
4. 近似現在状態・異未来対の区別
5. リスク深度誤差
6. seed間モード安定性
7. 特徴数の少なさ

単一の重み付き点数だけで決めない。

## 12. Holdout隔離

次をvalidationまでで固定する。

- 入力範囲
- 低次元数
- 力学系統
- マクロ特徴のみかTask 3特徴併用か
- 予測時間幅
- 警報閾値
- 警報継続条件

`task3_2_4_selection_lock.json`と検証ファイルが一致するまで、holdout状態ファイルを開かない。

## 13. アブレーション

holdoutで次を比較する。

- 選択候補
- マクロ特徴のみ
- 残差特徴なし
- 外力応答特徴なし
- Task 3公平再学習基準
- Task 3固定元モデル

候補比較表には分布のみ・全状態場、静的・DMD・DMDc・遅延残差の全候補を残す。

## 14. 成功判定

### A

公平再学習基準に対してholdoutで、次のいずれかを明確に改善し、検出率や誤警報を大きく悪化させない。

- 先行時間を2ステップ以上改善
- 検出率を0.10以上改善
- 近似現在状態・異未来対を0.15以上改善
- リスク深度誤差を20%以上改善

### B

核心改善には届かないが、深度誤差、対の区別、校正、残差警報、説明可能性の一部で改善する。

### C

複雑化しても単純基準を超えない、または核心性能を悪化させる。
