# Task 3.2-3 範囲
## 単純予測・早期警戒基準

## 1. 目的

Task 3.2-2で生成した連続軌道を使い、現在状態`X_t`と時点`t`までの履歴`L_t`だけで、高リスク構造をどこまで早く発見できるかを測る。

Task 3.2-3はTask 3.2-4のマクロ力学抽出が超えるべき比較基準を作るタスクである。

## 2. 主予測対象

- 未来時間幅内に高リスクへ入るか
- 高リスク開始までの警報先行時間
- 未来時間幅内の最大相対リスク深度
- 高リスク持続時間
- 軌道終了まで解除されない回復困難性候補

完全な`X_{t+1}`再現は主目的ではない。

## 3. 入力境界

正式入力:

```text
X_t + L_t
```

使用可能:

- 現在状態の分布要約
- 23状態配列の現在時点での分布加重要約
- 現在までの差分、傾き、加速度、変動、反転回数
- 時点`t`までに観測された外部入力
- 現在までの遷移記憶

入力禁止:

- scenario ID
- seed
- dataset split
- 将来状態
- 将来外部入力
- 将来イベント
- 正解ラベル
- 最終結果
- 不可逆性
- 同一seed通常軌道の未来情報

同一seed通常軌道は正解補正側だけで使用する。

## 4. 比較する5基準

1. 常時安全予測
2. 現在リスク閾値
3. 現在状態ロジスティック回帰
4. 直近傾向延長
5. 履歴ロジスティック回帰

リスク深度は各入力に対応したRidge回帰で補助予測する。

深層学習、木系高容量モデル、DMD、Koopman、HAVOK、VAMPは使用しない。

## 5. 履歴幅と予測時間幅

履歴幅:

```text
W = 2, 4, 8
```

予測時間幅:

```text
H = 4, 8, 16
```

警報継続条件:

```text
1, 2, 3ステップ
```

これらは探索用候補であり、普遍的な固定値ではない。

## 6. 正解生成

正解はscenario名から作らない。

対象軌道と同一seedの`stable_continuation`軌道を比較し、未来時間幅内で次を評価する。

- 相対リスクの持続上昇
- 集中増加・移動量低下・剛性上昇を伴う固定化候補
- 絶対リスクと相対リスクがともに深い崩壊候補

危険開始後の時点は早期警戒学習窓から除外する。

## 7. データ分割

```text
fit: seed 0, 1, 2
validation: seed 3
holdout: seed 4
```

行単位の無作為分割は禁止する。

同一seedと同一軌道は1つのsplitにだけ所属する。

## 8. Holdout隔離

次をvalidationまでで固定する。

- モデル系統
- 特徴量
- 履歴幅
- 予測時間幅
- 警報閾値
- 警報継続条件

固定内容は`selection_lock.json`へ保存する。

`selection_lock_validation.json`がlock hashを検証するまで、holdoutの状態・履歴ファイルを開くことを禁止する。

holdout後の設定変更は禁止する。

## 9. 評価

中心評価:

- 軌道単位の高リスク検出率
- 警報先行時間
- 誤警報率
- 適合率・再現率曲線下面積
- Brier score
- リスク深度平均絶対誤差
- リスク深度順位相関

時点行だけでなく、軌道単位の警報結果を必ず保存する。

## 10. Selection lock

validationの比較から次を固定する。

- 最良総合基準
- 最良現在状態基準
- 最良履歴基準

3基準をholdoutで評価するが、holdout結果から選択を変更しない。

## 11. 出力

- 特徴量schema
- 正解schema
- split manifest
- window manifest
- fit / validation結果
- 候補比較表
- selection lockと検証結果
- 選択モデル係数
- holdout結果
- 軌道別警報要約
- 適合率・再現率図
- 警報先行時間図
- 誤警報図
- リスク深度図
- 軌道別警報時系列
- Task 4引き渡し書

## 12. 完了条件

- 30軌道の正式探索データを使用
- 5基準を実行
- fit内seed交差確認
- validationで設定を固定
- holdoutをlock後にのみ読込
- 高リスク検出、先行時間、誤警報、深度順位を算出
- 正式Artifactを生成・再検証
- Task 4が使う最良単純基準を固定

## 13. 範囲外

- マクロ力学抽出
- 地形・循環・粘性・拡散・外力の分離
- 正式G_t・K_t
- 動的関係場
- ゲーム構造
- 不可逆性の証明
- 作用判断
- 作用モジュール接続
