# dept

DEPT / DEPT2 / H-DEPT research workspace.

This repository is a working research area for theory notes, validation code, tests, intermediate outputs, and old task files.

It is not a production-ready software package or a deployment-ready governance controller.

Stable reports, citation-ready releases, and cleaned reproducibility packages will be prepared separately for Zenodo.

## Current status

- Work in progress.
- Contains incomplete, superseded, and intermediate materials.
- Some files are historical task records and may not reflect the latest design.
- RC / Freeze files should be treated as the most stable references when present.

## Directory guide

- `docs/`: notes, task records, and design documents.
- `validation/`: validation and audit scripts.
- `tests/`: pytest-based checks.
- `localprep1/`: current integrated DEPT2 / action-module work area.
- `results/`: generated validation outputs; may be stale or incomplete.
- `.github/workflows/`: GitHub Actions workflows, mostly for manual validation.

## Note

This repository is mainly a research notebook and validation workspace. The cleaned public release should be read from the later Zenodo version, not from every intermediate file in this working repository.
