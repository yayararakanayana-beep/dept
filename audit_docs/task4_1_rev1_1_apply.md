# Task 3.2-4.1 Rev1.1 適用・再検証

## 目的

Rev1.1集中監査で確定した二つの修正を、凍結済みRev1の分岐台帳へ適用する。

1. 危険な縮小の再悪化条件を、全作用中の最大値から低費用成功経路群へ変更する。
2. C2条件付き脱出費用を、能動作用必要領域の判定と必要強度・残差費用の二段階モデルで再検証する。

## 実行方針

- Rev1の4,099分岐を再利用し、新規分岐は実行しない。
- seed 30～34の元軌道だけを決定論的に再生成し、Task 4特徴を復元する。
- PR #157で固定されたTask 4の26特徴をそのまま使用する。
- 予測候補の再比較・再選択は行わない。
- 元selection lockの検証後だけholdout特徴を生成する。

## 構造正解

成功分岐の最小費用を基準とし、絶対幅0.02または相対幅10%以内の成功経路を低費用成功経路群とする。この集合内の最大再悪化率だけを危険な縮小の再悪化証拠に使用する。

## C2費用予測

1. C2で能動作用が必要かをロジスティック回帰で判定する。
2. 作用不要領域は通常費用回帰を使う。
3. 作用必要領域は必要作用強度と、作用費用を除いた残差費用を別々に回帰する。
4. 予測強度から作用費用を再構成し、残差費用と合成する。

この構造は、費用正解を簡単化せず、鋭い作用境界を予測側で扱う。
